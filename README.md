This repository contains the source for [aidemd.dev](https://aidemd.dev) — the website for the AIDE methodology. It is not the tool you install and it is not where contributions are accepted.

AIDE (Autonomous Intent-Driven Engineering) cascades your intent through specialized agents into working, production-quality code. The full methodology is documented at [aidemd.dev](https://aidemd.dev).

## Looking for the tool?

The repo you want is the MCP server:

**[aidemd-mcp/server](https://github.com/aidemd-mcp/server)**

That is where AIDE lives as an installable tool for Claude Code, and where all issues, features, and contributions belong.

## This site

aidemd.dev is maintained by the core team. If you noticed a mistake on the site itself, open an issue here. For everything else — installing AIDE, contributing to the methodology, filing bugs — go to the server repo above.
